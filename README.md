
# 🚀 YouTube Spam Comment Detection using FastAPI & React

A Machine Learning web application that detects whether a YouTube comment is **Spam** or **Not Spam** using **Natural Language Processing (NLP)** and **Naive Bayes Classification**. The backend is built with **FastAPI**, while the frontend is developed using **React.js**.

---

## 📌 Features

- Detects Spam and Non-Spam YouTube comments
- Machine Learning model trained using Scikit-learn
- REST API built with FastAPI
- Interactive React frontend
- Real-time prediction
- Clean and modular project structure
- Swagger API documentation
- Easy deployment and scalability
